# ASMRDIY

ASMRDIY is an app that lets people create custom ASMR-style audio experiences from their own ideas and prompts. It was built for the Replit x ElevenLabs Buildathon using Replit for development and ElevenLabs for AI voice and audio generation.

## Demo

- Live app: [ASMR DIY on the App Store](https://apps.apple.com/us/app/asmr-diy/id6760271809)
- Demo: [Try the live demo on Replit](https://asmrdiy.replit.app)

## What the app does

ASMRDIY helps users create personalized ASMR experiences instead of relying on generic pre-made content. Users can layer ambient soundscapes, combine satisfying triggers, generate whispered AI voice audio, and create a more personal calming experience.

Examples:
- Create a relaxing ASMR session for sleep
- Build a focus-friendly sound experience
- Explore different calming styles, tones, and triggers
- Generate whispered audio from custom text prompts

## Built with

- Replit
- ElevenLabs
- Mobile app built in Replit
- ElevenLabs AI Whisper Voices
- Layered ambient sound and trigger mixing
- Session recording and replay
- Custom trigger recording

## How ElevenLabs was used

ElevenLabs was used for AI Whisper Voices, allowing users to type custom text and hear it played back in realistic whispered AI audio. This makes the app more interactive, personal, and aligned with the ASMR experience.

## How Replit was used

Replit was used to build, test, iterate, and publish the demo experience during development. It made it easier to prototype quickly and prepare the project for judges.
